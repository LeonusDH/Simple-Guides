# Шпаргалка:

### Создать новую screen сессию:

```bash
screen -S имя
```

### Свернуть сессию screen: `Ctrl + A + D`

### Просмотреть список запущенных сессий:

```bash
screen -ls
```

### Подключение по имени:

```bash
screen -x имя
```

### Подключение по номеру PID:

```bash
screen -x PID
```

### Закрыть окно или сессию: `Ctrl + A + K и подтвердить Y`
